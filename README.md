# Cloud SQL App

A Flask web application with an SQL backend, containerized using Docker and deployed on a Linux virtual machine in the cloud.

## Tech Stack
- Python (Flask)
- SQLite
- Docker
- Nginx
- Linux (Ubuntu)
- Git & GitHub
- Microsoft Azure

## Features
- Flask backend web application
- SQL database integration
- Dockerized for consistent deployment
- Reverse proxied using Nginx
- Designed and deployed on cloud infrastructure

```bash
python3 app/app.py
