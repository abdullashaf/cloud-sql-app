# Cloud SQL App

A simple Flask web application with an SQL backend, containerized using Docker and prepared for cloud deployment.

## Tech Stack
- Python (Flask)
- SQLite
- Docker
- Linux (Ubuntu)
- Git & GitHub

## Features
- Flask backend web application
- SQL database integration
- Dockerized for consistent deployment
- Designed for cloud environments

## Run Locally
```bash
python3 app/app.py
